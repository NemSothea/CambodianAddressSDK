# Third-Party Notices

This project itself has **no third-party code dependencies**. The notice below covers the
**bundled data** only.

## Bundled administrative dataset

`Sources/CambodiaAddressData/Resources/cambodia_address.json` is derived from the **pumi**
project, reshaped into this SDK's schema. pumi in turn compiles data from the NCDD (National
Committee for Sub-National Democratic Development) gazetteer of Cambodia
(<http://db.ncdd.gov.kh/gazetteer>).

- Project: pumi — https://github.com/dwilkie/pumi
- Data origin: NCDD Gazetteer of Cambodia
